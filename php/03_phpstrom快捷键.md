- PHPstorm默认快捷键
```
CTRL + D  复制当前行或复制选中内容
CTRL + Y  删除当前行或选中内容所涉及的行
CTRL + R  查找替换
ALT + 上下箭头 向上向下函数间切换
CTRL + Q  显示说明文档
CTRL + N  类名查找
CTRL + SHIFT + N  文件名查找
CTRL + SHIFT + ALT + N   函数名查找
CTRL + ALT + F12  跳转至当前文件所在磁盘上的位置
ALT + 左右键 标签切换
CTRL + F12 在当前类文件中快速查找方法
ctrl+alt+t        当前位置插入环绕代码
alt+insert        生成代码菜单
ctrl+f            查找
ctrl+r            替换
ctrl+g            跳转行
ctrl+alt+l        重新格式化代码
ctrl+shift+down   statement向下移动
ctrl+shift+up     statement向上移动
alt+shift+down    line向下移动
alt+shift+up      line向上移动
ctrl+/            行注释
ctrl+shift+/      块注释
Ctrl + Shift + F：区域查找
ctrl+shift+n      打开工程中的文件
ctrl+[]           匹配 {}[]
ctrl + '-/+': 可以折叠项目中的任何代码块
File->Export Setting 导出配置文件
File->Import Setting 导入配置文件
File->close Project->按delete建 关闭项目
File->Settings->Editor->General->Appearance->Show line numbers 显示行号
http://jingyan.todgo.com/shuma/2291272gjy.html 配置phpStorm主题
font 16 间隔1.2 比较舒服
```

- 卸载重新安装，避免过期日期
```
0. 保留配置文件
1. 卸载安装包 - 从控制面板卸载
2. 重新安装 - 重新导入配置文件
```
