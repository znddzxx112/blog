### 安装从U盘引导

```

install centos7 按e

修改 hd:/dev/sda4 后 按 ctrl + x


```

### 分区
```
fdisk /dev/sda4

d // 删除
p // 打印
n // 新建分区,1,回车
w // 保存退出
```

### 格式化
```
mkfs.etx4 /dev/sda4
```

### 挂载
```
mount -t ext4 /dev/sda4 /data1
```

### 开机自动挂载
```
vi /etc/fstab

/dev/sda4 /data4 ext4 defaults 0 0
```
