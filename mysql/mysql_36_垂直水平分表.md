- 水平逻辑
```
 逻辑：0、用户相关表进行路由 index= ceil(abs(userid) / 1000w)1000w数据一张表
 0.1 维护用户相关的一组分表,该用户会落在哪张表中
 用户所在表最小userid:(index - 1) * 1000w + 1
 用户所在表最大userid:index * 1000w
 逻辑：1、使用路由表
 比如：
 user_post_index_\d，user_info_\d
 1、解决场景
 1、 添加用户数据（单条），根据表路由，增加即可，sql返回1146错误，则创建表后再添加
 2、 更新用户数据（单条），根据表路由，更新即可
 3、 获取用户列表（单条），根据表路由，获取即可
 4、 获取用户列表（多条），根据表路由，将多条语句，使用union all获取即可
```
